# Inventory Icon Requirements

These icons are used by the inventory UI.

Location: `assets/textures/ui/inventory/`
Format: 64x64 PNG with transparency (RGBA)

| food.png | Food consumable |
| coin.png | Currency |
| wood.png | Wood resource |
